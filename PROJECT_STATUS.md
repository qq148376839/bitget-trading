# 项目状态

**更新时间**: 2026-02-08

## 项目概述

Bitget 加密货币量化交易系统 — 基于 Bitget API 的合约 Maker 剥头皮自动化交易平台。

## 当前状态：核心功能完成

### 已完成
- 项目目录结构
- Claude Code agents（10 个）
- 后端 API 基础骨架（现货 + 合约）
- Bitget API 客户端封装
- 合约 API 服务（行情、订单、账户）
- 剥头皮策略引擎（状态机 + 双循环）
- 风控系统（回撤限制、日亏限制、仓位限制）
- 挂单合并引擎
- 策略控制 REST API（8 个端点）
- 前端策略仪表盘（Next.js 14 + Ant Design 5）
- Docker Compose 部署（nginx + api + frontend + postgres）
- 数据库迁移脚本
- 项目文档体系

### 待开发
- WebSocket 实时数据推送
- 回测系统
- 多策略支持
- 策略历史数据持久化
- 单元测试 / 集成测试
