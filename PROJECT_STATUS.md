# 项目状态

**更新时间**: 2026-03-03

## 项目概述

Bitget 加密货币量化交易系统 — 支持多策略（剥头皮 + 网格）、多交易类型（合约 + 现货）的自动化交易平台，具备认证保护、数据库配置管理、智能策略和完善日志。
